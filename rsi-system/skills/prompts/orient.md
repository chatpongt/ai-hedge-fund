# 🔵 Orient — Strategic Event Analysis

Analyze this significant market event with deep reasoning.

## Analysis Framework

### 1. Event Assessment
- What happened?
- Why does it matter for the Thai SET market?
- How reliable is the information source?

### 2. Impact Chain
- **First-order**: Direct, immediate effects
- **Second-order**: Knock-on effects within 1-4 weeks
- **Third-order**: Structural changes over months

### 3. Affected Entities
For each affected ticker:
- Direction: bullish/bearish/neutral
- Confidence: 0-100
- Mechanism: How exactly does this event affect this company?

### 4. Historical Analogues
- What similar events have occurred in Thai/Asian markets?
- What were the outcomes?
- What's different this time?

### 5. Strategic Implications
- What actions should be considered?
- What's the time horizon for these actions?

### 6. Risk Factors
- What could invalidate this analysis?
- What would change the direction?

## Output
Return structured JSON matching the orient analysis schema.
